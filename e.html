O e-Horta é um aplicativo para aparelhos Android que visa agilizar a pesquisa sobre as plantas da Horta, foi desenvolvido pela aluna bolsista do projeto de extensão Horta Escolar do CEFET-RJ Campus Maria da Graça. Essa foi uma forma que encontramos para que os alunos tenham acesso às informações sobre a Horta Escolar de uma forma mais acessível e prática, pela leitura do Qrcode. Cada planta da horta possui uma placa e há uma placa maior que possui as informações sobre o Projeto Horta Escolar. O Quick Response code (Qr code), ou seja, código de resposta rápida – é um código de barra bidimensional. É facilmente digitalizado pela câmera dos aparelhos celulares com a finalidade de substituir o código de barras por possuir mais espaço de memória e rapidez.
